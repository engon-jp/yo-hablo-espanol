# ¡Soy estudiante! — SER 作文練習アプリ

## 概要
動詞 **ser** と主語人称代名詞の作文練習アプリ。既存の「¡Yo hablo español!」と同じシリーズの姉妹アプリで、見た目・操作感（水彩絵本×Duolingo風・歩く本マスコット・タラベラタイル装飾・ボトムシート選択・全画面参照ビュー・フラッシュカード）を踏襲し、中身を ser 用に差し替えたもの。学生にスマホ配布する前提。

## 基本情報
- **形態**：単一 `index.html`（ゼロビルド・依存なし・vanilla JS）。PWA対応（`manifest.json`）。
- **親ランチャー**：`~/Documents/_MyProjects/アプリ制作/index.html`（仮タイトル「Papelito Español」）から `yo-hablo-espanol/` と `soy-estudiante/` を選択できる。各アプリのヘッダーに「選択」へ戻るボタンを追加済み。
- **保存キー**：localStorage `ser-trainer-v1`（レベル・ラウンド・スコア）。
- **1ラウンド**：20問（その場自動生成）。レベル別に独立したラウンド。
- **音声**：文はその場生成のため端末の音声合成（Web Speech API, `es-ES`）で再生。主語人称代名詞表では `assets/audio/pronouns/singular.mp3` / `plural.mp3`、ser活用表では `assets/audio/conjugation/ser.mp3` を再生（失敗時は Web Speech API で代替）。文のMP3化は未使用（将来ElevenLabs等で追加＝任意）。読み上げ文字列は `currentSpeakText()` が `state.current.es`/`.esQ`/`.esA`（＝生成済みの正文）をそのまま読むだけなので、文生成ロジック側の修正（大文字化など）は自動で音声側にも反映される。UI文言（placeholder等）の変更は音声には無関係。
- **判定**：Nivel 1〜3 は厳格一致（大小文字・アクセント・記号をそのまま比較）。Nivel 4 は自己採点。

## 出題方式についての設計判断（2026-07-01）
- **固定文リストではなく、文法ルールに基づくその場生成**。yo-hablo-espanol（48文を人手で作成・校正・録音）とは方式が異なる。
- 生成しうる文の総数：**Nivel 1/2（単文）= 14主語 × 6補語 × 3パターン(肯定/否定/疑問) = 252通り**、**Nivel 3/4（一問一答）= 12主語 × 6補語 × 2(はい/いいえ) = 144通り**。無限ではなく上限のある組み合わせ空間（合計約396通り）。
- 採用理由：ser は主語14種×補語の性数一致×肯定/否定/疑問の組み合わせが多く、48文方式のように全パターンを人手で書いて校正するのは非現実的。そのためルールをコード化し、ルール自体を大量サンプリング（8,000回のランダム生成）でテストする方式に切り替えた。
- **遠藤さんと協議の上、固定リスト化はせず現状の生成方式を継続する方針に確定**（2026-07-01）。理由：ser の練習は「毎回ルールを当てはめ直す」訓練であり、同じ文が出ても反復練習として有効。固定リスト化最大のメリット（ElevenLabs等での自然な録音音声）よりも「発音練習より文法練習」という位置づけと、作業の手間を優先。
- 副産物：今回作った生成ロジックは、将来もし固定リストに切り替えたくなった場合の「候補文の大量生成→人が選定」というたたき台作成ツールとしても再利用できる。

## 出題仕様
- **Nivel 1**：3つの選択枠 ①主語 ②ser ③補語（否定は②前に固定`no`枠）。①②は4択、③は補語により2択(学生)/4択(先生・〜人)。
- **Nivel 2**：①②③すべて手入力。③補語はタップでヒント（Nivel 1と同じ選択肢）。否定は②前に空の`no`入力。
- **Nivel 3**：一問一答。主語が1人称・2人称なら西語の主語は省略、3人称は疑問文のみ主語表示・答えは省略。ser・はい/いいえ・隠れ`no`・3人称主語はヒント無し手入力、補語はタップでヒント。
- **Nivel 4**：Nivel 3と同じ内容を「質問文／答え文」2つの入力欄に全文手入力→正解表示→自己採点。
- **補語**：学生 estudiante(s)／先生 profesor(a)(es)(as)／国籍4種（日本人 japonés系・スペイン人 español系・フランス人 francés系・中国人 chino系）。
- **参照**：左下「主語人称代名詞」表（単数・複数の音声つき）、右下「serの活用」表（音声つき）。単語チェックボタンはレベルタブ下。
- **単語チェック**：学生・先生＋各国の 国名/〜人(男女単複4形)/〜語 を指定順で。**日→西のみ**（西→日は表示のみ・選択不可）。
