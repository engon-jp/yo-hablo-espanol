# スペイン語 文法トレーナー — PROGRESS

## 概要
初学者向けに、スペイン語の動詞活用と語順（肯定・否定・疑問）を練習する学習アプリ。
全48問の固定マスターフレーズをベースに、Nivel 1〜4 で出題形式を変える。
仕様確定版：`/Users/endokenta/Downloads/gemini-code-1782625117911.md`

## 基本情報
- 形式：ビルド不要の**単一HTMLファイル**（`index.html` に HTML/CSS/JS・48問データ・全ロジックを内包）。依存ライブラリなし。
- 音声：`assets/audio/phrase_01〜48.mp3` を再生。活用表では `assets/audio/conjugation/*.mp3` を再生。**未配置の間は端末の音声合成（Web Speech API・es-ES）で自動代替**。mp3 を置けば自動でそちら優先。
- 対象：学生に配布し、スマホでも使える（レスポンシブ／アクセント補助バー付き）。

## 設計メモ（実装の要点）
- **文パーサ** `parsePhrase`：全48問が「[¿] 主語 [no] 動詞 目的語 [./?]」の規則構造 → 自動分解して各レベルの空欄を生成。
- **規則活用** `conjugate`：9動詞すべて規則（-ar/-er/-ir）。Nivel 1 の動詞4択・活用表を動的生成。
- **厳格判定** `strictEqual`：大小文字・アクセント・`¿?!.`・`no` をそのまま比較（前後空白のみ無視）。Nivel 2/3 共通。
- **Nivel 4 柔軟判定** `judgeLevel4`：yo/tú/nosotros(as)/vosotros(as)は主語省略・語順変化を正解扱い。3人称は主語必須。文末記号・アクセントは厳格。
- **誤選択肢**：主語＝12種プールから（nosotros/nosotrasペアは同時出現不可）／動詞＝同動詞の他人称のみ／目的語＝動詞別プール4語を毎回シャッフル。
- **出題**：各レベル独立の25問ラウンド。全48問からランダム抽出、消化後に再抽選。`SAVE_KEY=es-trainer-v2`。
- **データ修正**：仕様書 #36 の誤記 `Ellas write...` を `Ellas escriben un mensaje.` で登録済み。男性複数（Nosotros/Vosotros）問題に「（男性）」明記済み。

## 実装済み機能（2026-06-30時点）

### UI・デザイン
- 水彩絵本タッチ × Duolingo的ポップ × スペイン/ラテンアメリカ世界観
- タイトル「¡Yo hablo español!」、kicker「Gramática」、サブ「主語人称代名詞＋規則動詞の作文練習」
- papel picado風バンティング（上部固定）、街並みシルエット（下部）、タラベラ風タイル帯（問題欄上部）
- SVGベレー帽マスコット（仮）：正解で笑顔＋バウンス＋紙吹雪、不正解で困り顔＋汗
- 丸く厚い3Dボタン（押下で沈む）、進捗バー、⭐スコア（累計）

### 出題ロジック
- **各レベル25問**（全48問からランダム抽出、レベル独立）。`SAVE_KEY=es-trainer-v2`、`ROUND=25`
- 25問消化で新たに25問を再抽選（毎回違う問題セット）
- 進捗・スコアは localStorage 保存（リロードで再開）

### Nivel別仕様
- **Nivel 1**：主語（「主語」ラベル）・動詞（不定詞表示）をタップ→4択。同動詞の活用形のみ。nosotros/nosotrasなど性別対が同時に並ばない排他処理。noは確定表示。未選択で解答しても不正解として次へ進める。
- **Nivel 2**：主語・no・動詞を手入力。動詞欄に薄字で不定詞ヒント。文頭自動大文字化。
- **Nivel 3**：主語・no・動詞は手入力、最後の空欄はタップ4択（動詞別プールを毎回シャッフル）。未選択で解答しても不正解として次へ進める。
- **Nivel 4**：日本語のみ表示→全文手入力。**柔軟判定**：yo/tú/nosotros(as)/vosotros(as)は主語省略可・語順変化も正解、3人称は主語必須。
- 「← 前」ボタン：前の問題の正解を確認できる（スコアに影響なし、pos=0はdisabled）

### 参照ビュー（活用表・人称代名詞表）
- **Nivel 1/2**：「📋 主語人称代名詞」「📖 ○○の活用」ボタンを常時表示
- **Nivel 3/4**：解答後に両ボタンが表示される
- 活用表：9動詞すべて `conjugate()` から動的生成、語幹＋語尾（語尾は下線強調）、PDFと同じレイアウト
- 活用表音声：各動詞の活用表に音声ボタンを表示し、`assets/audio/conjugation/{不定詞}.mp3` を再生（失敗時は Web Speech API で代替）

### 単語チェック（フラッシュカード）
- 全37語（動詞9 + 名詞28）を収録。[`VOCAB`配列 in index.html]
- 西→日 / 日→西 モード切替
- 順番 / ランダム 切替（37語すべてを網羅）
- カードタップで答え表示。名詞は [男][女][固有] タグ付き、動詞はタグなし
- 日→西モードでは日本語の上に品詞ラベル「（動詞）」「（名詞）」を表示（斜体なし）
- 西→日モードでは品詞ラベルはめくった後の答え部分に表示（ネタバレ防止）

### その他
- アクセント補助バー（á é í ó ú ñ ¿ ? .）
- 音声：mp3優先 / なければブラウザ内蔵音声合成（es-ES）にフォールバック
- 「解答する」「正解はこちら」「特殊文字と記号の入力はこちら」等の文言確定済み
- 正解時も不正解時と同様に正解文と日本語訳を表示。
- 「私たち/君たち」の男性複数（Nosotros/Vosotros）に「（男性）」を明記し女性形と統一
